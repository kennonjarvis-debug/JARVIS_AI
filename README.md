# JARVIS_AI

Consolidated monorepo for JARVIS_AI projects.

## Structure

- `web/` - Web applications
- `api/` - Backend services
- `shared/` - Shared libraries and utilities
- `docs/` - Documentation

## Getting Started

```bash
# Install dependencies
pnpm install

# Run all apps
pnpm dev

# Build all projects
pnpm build
```
